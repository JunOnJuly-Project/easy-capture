## 3. 메인 윈도우 (세그 → 선택 → 범위)

```
┌──────────────────────────────────────────────────────────────┐
│ easy-capture · GIF 모드        파일: mv.mp4        _ □ x        │
│ ───────────────────────────────────────────────────────────  │
│ ┌───────────────────────────────────┐ ┌────────────────────┐ │
│ │                                   │ │ 검출된 오브젝트      │ │
│ │      [ 영상 프레임 캔버스 ]        │ │ ◉ person 1  (선택)  │ │
│ │                                   │ │ ○ person 2          │ │
│ │   ┌─────┐  ← 클릭한 오브젝트 마스크 │ ○ person 3          │ │
│ │   │ 👤  │     + 크롭 범위 박스(드래그)│────────────────────│ │
│ │   └─────┘                         │ │ 크롭 범위           │ │
│ │                                   │ │  W [ 540 ] H [ 960 ]│ │
│ │                                   │ │  종횡비:◉9:16 ○1:1  │ │
│ │                                   │ │       ○16:9 ○자유   │ │
│ │                                   │ │ 안정화: ○약 ◉표준 ○강│ │
│ │                                   │ │────────────────────│ │
│ │                                   │ │ 업스케일 □ 사용      │ │
│ │                                   │ │  모델 [Real-ESRGAN▾]│ │
│ │                                   │ │  배율 ◉2x ○4x       │ │
│ └───────────────────────────────────┘ └────────────────────┘ │
│ 타임라인                                                       │
│ │◀─[시작]══════════▮(현재)═══════════[끝]─▶│  00:12 / 03:40   │
│        ▲ 드래그 핸들              ▲ 드래그 핸들                 │
│ [세그 실행] [추적 실행]                  진행률 [░░░] ETA --     │
└──────────────────────────────────────────────────────────────┘
```

- 좌: 프레임 캔버스(마스크 오버레이 + 크롭 박스 드래그).
- 우: 검출 후보 목록(다중 인물 라벨), 크롭/종횡비, 안정화 강도, 업스케일.
- 하단: 타임라인. **시작/끝 드래그 핸들 + 현재 시점 마커**.

> **툴팁/안내 (비전문가 배려)**
> - **안정화**: "오브젝트가 프레임마다 떨리지 않도록 크롭 위치를 부드럽게 보정합니다" (약/표준/강 = 이동평균 윈도우 3/5/10).
> - **업스케일 사용 시 경고 배너**: "⚠ 업스케일을 켜면 프레임 간 약간의 떨림(플리커)이 보일 수 있어요. 프레임 일관성 보정은 v1.1 예정."
> - **종횡비**: "9:16 = 세로(릴스·쇼츠), 1:1 = 정사각(인스타), 16:9 = 가로".

## 4. 교정 모드 (수동 부분 재추적)

```
┌──────────────────────────────────────────────────────────────┐
│ 미리보기 · 교정 모드                          00:21 (frame 631)│
│ ───────────────────────────────────────────────────────────  │
│ ┌───────────────────────────────────┐                         │
│ │   [ 일시정지된 프레임 + 세그 재오버레이 ]                     │
│ │   ○ person 2 ← 잘못 추적됨(현재)                             │
│ │   ◉ person 1 ← 여기를 클릭해 재지정                          │
│ └───────────────────────────────────┘                         │
│ │═════════✓ 성공 구간 ═════════▮━━ 재추적 대상 ━━━━━━━━│       │
│                              ▲ 이 지점부터                      │
│ [올바른 대상 클릭] →  [ 이 지점부터 재추적 ]   [취소]           │
└──────────────────────────────────────────────────────────────┘
```

- 이전 성공 구간(`✓`)은 보존, 현재 프레임(`▮`) 이후만 재추적.

## 8. 컷 재매칭 확인 다이얼로그 (추적 중)

샷 경계 재매칭 점수가 임계값 미만일 때 표시(자동 진행 방지).

```
┌──────────────────────────────────────┐
│ 컷 이후 같은 인물 재확인              │
│ ────────────────────────────────────│
│  [ 직전 캡처 썸네일 ]  →  [ 현재 후보 ]│
│     person 1                후보 A    │
│                                      │
│  같은 사람이 맞나요?                  │
│  [아니요·다시 선택]  [건너뛰기]  [맞음]│
└──────────────────────────────────────┘
```

- "맞음" → object_id 유지·추적 계속. "다시 선택" → 교정 모드(§4). "건너뛰기" → 해당 구간 종료(갭 정책).
